 # How to Install
 
 1. Navigate to the folder DB
 
 2. Use mysql -u (username) -p
 
 3. Run source schema.sql
 
 4. Open your another terminal and navigate to the develop folder and run   npm run seed   .
 
 
 
 # Usage
